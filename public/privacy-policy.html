Privacy Policy

Last updated: 30-11-2025

RivInspect is a client-side web tool for previewing and inspecting .riv animation files. It is designed with privacy in mind, and this website does not collect, store, or transmit any personal data.

File Processing

  • All .riv files are processed locally in your browser.
  • Your files are never uploaded to any server.
  • No file data is stored, transmitted, or logged by RivInspect.

No Data Collection

RivInspect does not:

  • collect analytics or usage metrics,
  • use cookies or tracking technologies,
  • log IP addresses or device identifiers,
  • store any personal information.

Hosting

This site is hosted using GitHub Pages, which serves static files only and has no server-side processing. GitHub may collect limited, non-personal technical logs as part of their hosting platform. You can review GitHub's privacy statement here:

GitHub General Privacy Statement
