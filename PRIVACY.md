# Privacy Policy

Last updated: 30-11-2025

RivInspect is a client-side tool that runs entirely in your browser.  
This project does **not** collect, store, or transmit any user data or files.

## File Processing
- All `.riv` files you load into RivInspect are processed **locally in your browser**.
- Your files are **never uploaded** to any server.
- No file contents are logged, transmitted, or stored by RivInspect.

## No Data Collection
RivInspect does not:
- collect analytics or usage data,
- use cookies or tracking technologies,
- log IP addresses,
- store any personal information.
